# GIF Finder

A fast, modern GIF search application built with Vanilla JavaScript and the GIPHY API. Search, preview, download, share, and organize your favorite GIFs with a clean, responsive interface.

## ✨ Features

### 🔍 Smart Search

* Instant GIF search
* Trending GIFs
* Search suggestions
* Recent search history
* Category shortcuts
* Infinite scrolling

### 🎨 Modern UI

* Responsive design
* Glassmorphism interface
* Smooth animations
* Skeleton loading
* Sticky search header
* Dark theme

### 🖼️ GIF Preview

* Fullscreen preview modal
* GIF information
* Creator details
* Dimensions and file size
* Related GIF recommendations
* Keyboard support (Esc to close)

### ❤️ Favorites

* Save favorite GIFs
* Local storage support
* One-click favorite toggle
* Persistent across sessions

### 📥 Download & Share

* Download GIF
* Download MP4 (when available)
* Copy GIF URL
* Copy Markdown
* Copy HTML embed code
* Native sharing support

### 🚀 Performance

* Lazy-loaded images
* Infinite scrolling
* Skeleton placeholders
* Optimized API requests
* Fast rendering
* Responsive layout

## ⌨️ Keyboard Shortcuts

| Key     | Action            |
| ------- | ----------------- |
| `/`     | Focus search      |
| `Enter` | Open selected GIF |
| `Esc`   | Close preview     |
